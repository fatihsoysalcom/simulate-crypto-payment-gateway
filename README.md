# Simulate Crypto Payment Gateway

This example demonstrates a client-side simulation of initiating a cryptocurrency payment for e-commerce products. It showcases how product selection can trigger a simulated API call to a crypto payment gateway, displaying placeholder payment details like amount, address, and network to the user.

## Language

`html`

## How to Run

Open index.html in your web browser.
